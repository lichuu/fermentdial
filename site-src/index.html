FermentDial

Fermentation temperature controller firmware for the M5Stack Dial. Install it from this page over USB — no toolchain required. After flashing, the installer asks for your Wi-Fi network and hands you a link to the device dashboard. Wi-Fi credentials travel over the USB cable only; this site never sees them.

Demo firmware

Simulated sensor, all outputs disabled — safe way to explore the UI and dashboard with nothing wired up.

Web flashing needs Chrome or Edge on desktop. Web flashing only works over HTTPS.

Real-sensor firmware

DS18B20 probe + heater/pump control. Wire the hardware first — see the wiring guide.

Web flashing needs Chrome or Edge on desktop. Web flashing only works over HTTPS.

Notes

  • Use a USB-C data cable, not a charge-only cable.
  • Works in Chrome and Edge on desktop (Web Serial). Firefox and Safari can't flash — use the PlatformIO route instead.
  • If no serial port shows up, hold the Dial's side button while plugging in USB to enter download mode.
  • Skipped the Wi-Fi step? The device starts an open FermentDial-Setup-XXXX access point — join it and open http://192.168.4.1/.
  • Future updates install from the dashboard: Settings → System → Check for updates.
